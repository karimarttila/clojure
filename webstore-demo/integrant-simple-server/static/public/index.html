Simple Server

This is the Simple Server. Available commands are:

  • GET /info - Gives general information.
  • GET /index.html - This file.
  • POST /sign-in - Creates a new user.
  • POST /login - Logs on to service using credentials created using sign-in. You get json web token that must be supplied in following requests.
  • GET /products-groups - Gets product groups.
  • GET /products/ - Gets products in product group.
  • GET /product/ - Gets specific product information.
NOTE: You can see examples how to call these endpoints in the scripts directory (using curl).
